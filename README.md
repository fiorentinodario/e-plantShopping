# Paradise Nursery Shopping Cart

Paradise Nursery is a React and Redux shopping cart application for an online houseplant store.

The application includes:

- A landing page with company information and a Get Started button.
- A product listing page with categorized houseplants.
- Add to Cart functionality with disabled buttons after selection.
- A dynamic navigation bar with a cart icon and item count.
- A shopping cart page with quantity controls, item totals, total cart amount, delete buttons, Continue Shopping, and Checkout functionality.

This project was created for the IBM React final project and is stored in the public GitHub repository `e-plantShopping`.
